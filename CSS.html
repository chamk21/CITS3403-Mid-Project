History

CSS took many years to appear in the world. The first ever websites had no way of differentiating from each other due to the fact that they all looked very similar. There were many initial concepts of styling. One such was from Pei-Yuan Wei, who created the graphical ViolaWWW Browser in 1991. He incorporated his own stylesheet language right into his browser.

Another approach was from Marc Andreessen in 1994. He was the founder behind both the Mosaic and Netscape browsers. His extended HTML to include tags that allowed for presentational and visual features. However this failed because it wasn't much of an improvement from ViolaWWW browser.

It meant that styles can inherit and overwrite other styles that had previously been declared following a fairly complicated hierarchy known as specificity. This also meant web developers were allowed multiple stysheets in one page. This was revolutionary as it gave web developers a chance to make their websites stand out from each other.

Seperating content from style

Separation of style from content is crucial in web design. For large websites its easier to open the CSS page corresponding to that page and make changes. It's much easier than searching for style attributes in a large HTML file epically when there could be a combination of inline and internal CSS in the file. This separation helps with consistency by making it easier to maintain sites and share style sheets across pages. Another advantage of moving CSS styles to separate file is cacheability. Browsers can cache style sheet file on client side, thus saving bandwidth and reducing loading time.

Selectors

To apply CSS we have to use selectors. CSS selectors are used to "find" (or select) the HTML elements you want to style. We can divide CSS selectors categories.

Simple selectors:These are selectors that select using id,class or certain element e.g #id and .info and img{}

Combinator selectors: A combinator is something that explain the relationship between the selectors. An example* of a combinator selector is "li a". This is all the anchor tags inside the list item

Pseudo-class selectors: CSS classes used to define the state of an element. An example* is "li a:hover:not(.active)" This allows certain states such as hovering over a link to be selected.

Pseudo-elements selectors: A pseudo-elemet allows for a specific part of the element to be styled
p::first-line { color: blue; text-transform: uppercase; }
The first line of every p element has the color blue and text uppercase applied

Attribute selectors:This allows to select elements based on an attribute or attribute value. For example a[value="password"] allows for all anchor tags with value password to be styled.

Note that examples marked with * are example taken directly from this page. You can see this by right clicking and inspecting element.
